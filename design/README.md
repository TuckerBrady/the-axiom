# The Axiom — Design Artifacts

This folder contains all approved visual design artifacts for 
The Axiom. HTML prototypes must be approved by Tucker Brady 
before the designs are implemented in React Native.

## screens/

HTML prototypes for individual screens. Open in any browser 
to preview. These are the source of truth for visual 
implementation.

| File | Screen | Status |
|------|--------|--------|
| boot-initial.html | Initial boot log (onboarding) | Approved |
| boot-return.html | System boot log (returning player) | Approved |

## Adding new prototypes

1. Build the prototype as a standalone HTML file
2. Get Tucker sign-off
3. Add to the appropriate subfolder
4. Update this README with the new entry
5. Reference the file path in any Claude Code implementation prompt
